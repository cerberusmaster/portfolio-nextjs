## About

Capiwise is Trading App. People want to be traders and all traders want to know immediate changes of currencies, stocks, metals and this is what Capiwise do.
Capiwise watches stock/currency changes in real time and send push notifications and emaiil alerts based on conditions users created.
We worked on [Git](https://github.com/capiwise-open), [Codemagic](https://codemagic.io/) for automatic deployement

## Contribution

- Implemented React State Management with API integration based on Redux, React Query, React Toolkit modules
- Integration with third-party APIs like [Twelvedata](https://twelvedata.com/docs#getting-started) for stock, and cryptocurrency changes
- Added linear chart, circular chart components that shows stock changes

- Upgrade project into **AWS Amplify** for efficient optimized AWS endpoints management
- Build **Serverless API** with Nodejs based **Lambda functions** and AWS API Gateway
- Implemented Authentication (Google, Apple, Email/password) with **AWS Cognito**
- Implemented **Push notification**, **Email notification** with FCM (Expo Push Notification), AWS SNS, and AWS SQS
- Added AWS Event bridges/Schedule that checks stock changes and daily news notification
- Improved performance of API response and content loading with **AWS Redis**
- Setup **AWS VPC** for network security
- AWS RDS with MySql, PostgreSql

- Developed Content Management System with [Strapi](https://strapi.io/) (headless CMS), built schema, and deployed into AWS EC2
