## About

[Master Anesthesia](https://masteranesthesia.com) provides contents of anesthesia, drugs, community of nurses, and education course for students and anesthetists.
I take responsibility in all aspects of development in this project. I helped [Matthew](https://www.facebook.com/matthew.willis.798/) to implement business requirements and upgrades application to best performance.
Working with Matthew was great experience and it wouldn't be removed in my career

## Mobile app

* Upgraded react native and expo modules to latest and solved compatibility issues happened for [iOS]((https://apps.apple.com/us/app/master-anesthesia/id1550793078#?platform=iphone)) and [Android](https://play.google.com/store/apps/details?id=com.masteranesthesia)
* Optimized interaction with firebase firestore and realtimedb and speed up loading contents from 15min in a few seconds
* Implemented **firebase authentication** with social providers email/password, google, apple, facebook and anonymous
* Enhanced code quality by transfering javascript to typescript and adding test codes

## Admin dashboard

Built admin [dashboard](https://masteranesthesia.vercel.app) using [Primereact](https://primereact.org/) that manages parameters of applications, contents, credits for courses, content management
Developed firebase cloud functions that validate authentication, user's progress, contributions etc

## WordPress

* Integrated WordPress and firebase authentication with users using [Integrate Firebase PRO](https://firebase-wordpress-docs.readthedocs.io/en/latest/) plugin
* Added education courses for mastering anesthesia using [Memberpress](http://memberpress.com/)
* Customized plugin and theme according to provided design
* Integrate stripe with memberpress webhook and actions
* Synchronization **Firestore**, **Realtimedb** and **MySql** by actions, webhooks and filters
* Built **AI chatbot** and direct contact to owners using OpenAI modules and [Intercom](https://www.intercom.com/)
